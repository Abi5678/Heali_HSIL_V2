# Audio assets

This folder holds audio files used by the app.

- **companions/** — Heali voice preview WAVs for onboarding (hover-to-play).
  - Generated by: `uv run python scripts/generate_heali_voice_previews.py`
  - Files: `heali_balanced.wav`, `heali_calm.wav`, `heali_energetic.wav`, `heali_informative.wav`
